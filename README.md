## Installation
```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## ⚙️Settings

Specify settings in _main.py_
* `USE_PROXY` - True | False
* `SHUFFLE_WALLETS` - True | False
* `SLEEP_BETWEEN_WALLETS` - 2 values in seconds
* `REF_POOL` - ["YOUR_REF1", "YOUR_REF2"] etc.
